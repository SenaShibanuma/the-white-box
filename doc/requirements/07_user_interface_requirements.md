# 07. ユーザーインターフェース要件 (Observer Console UI)

観察者は、16GB RAMの負荷を抑えつつ、シミュレーションの全パラメータをリアルタイムで監視・操作できるダッシュボードを持つ。

## 1. 開発フレームワーク
- **Streamlit (Python)**: 軽量かつデータ可視化に優れ、16GB RAM環境でも動作が軽快。

## 2. 監視機能 (Real-time Monitoring)
- **対話ログ**: A/Bの発言を色分け（例: A=青, B=緑）して表示。
- **情動ステータス**: 
  - 4象限レーダーチャート（適合・共鳴・摩擦・減衰）。
  - 各エージェントの「情動の推移グラフ」。
- **駆動パラメータ**: 
  - 飽和度 (Satiety) と 好奇心 (Novelty) のプログレスバー。

## 3. 設定・介入機能 (Control Panel)
- **人格設定**: 起動前およびリセット時に `06. 人格ライブラリ` から2名を選択可能。
- **動的パラメータ調整**:
  - 減衰係数 ($\gamma$) のスライダー調整。
  - 禁忌ワードリストの動的追加・削除。
- **マニュアル介入**:
  - **強制沈黙**: 1ターン発言をスキップさせる。
  - **強制忘却**: SQLiteから特定のキーワードに関連する記憶を即座に抹消する。
  - **監視者ステート切替**: `Active` / `Asleep` を手動でトグル切り替え。

## 4. セーフティコントロール
- **緊急停止ボタン**: シミュレーションを即座に中断し、現在のSQLite DBと全ログを `.zip` でエクスポートする。
